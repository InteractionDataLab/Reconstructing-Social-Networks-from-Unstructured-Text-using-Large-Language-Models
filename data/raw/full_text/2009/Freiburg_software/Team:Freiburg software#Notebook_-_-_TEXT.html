Here is one example: Key features of this change: multible upload fields in forms There is a submitting sequence: the files are submitted one after the other. Then the form is submitted. Many events are fired: upload started and completed (for each file), on form submit The changeValue event of the form contains upload infos such as filenames The files are send to the target url speciefied in the item structure.
