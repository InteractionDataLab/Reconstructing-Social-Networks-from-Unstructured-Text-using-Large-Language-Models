We decide to make 4 file containers to hold different kind of contents: global : contains unit definition, definition of standard conditions of cultivating bacterial, and so on. module : module of each species, including biobricks, virtual biobricks, cells, plasmids, substituents and so on. species : each species that occurs in reactions defined in reaction container is defined here. reactions : reactions of all species defined in species container are defined here.
