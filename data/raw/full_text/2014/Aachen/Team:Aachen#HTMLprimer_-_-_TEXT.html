Including Floats into your text Including floats (figures, tables, etc.) into HTML unfortunately is not as straight forward as, for instance, in \LaTeX. Let's start with the least complicated stuff, and then get more involved. Therefore first we're linking websites, then including images, then tables and figures. Linking (other) websites In HTML there's no distinction between linking to some place on your own website, or to a foreign website. Everything uses the same construct (thank god!) which starts with . text The address is either the name of an object (header, paragraph, block, ...) or the URL of a website, for instance https://2014.igem.org/Team:Aachen/. If you link to any of our sub-pages, please use the full URL. The text within the tag is the clickable link. You should keep that as short as possible, but as long as necessary! Therefore a link to our start page would look like: link If you would like to specify the destination of the link, eg. loading in the same frame/windows/tab or in a new tab/window, you can choose between _blank for a new tab/window or _self for the current frame/window/tab. link Tables Creating tables in HTML is a tedious task.
