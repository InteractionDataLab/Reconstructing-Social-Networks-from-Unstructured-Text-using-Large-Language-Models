Notebook / Locate GFP and the CBD

Here we list all the steps that were required to locate the cellulose binding domain (CBD) within the cell using GFP as a marker.ONC = overnight culture ; CBD = Cellulose binding domain ; IPG = Institute for plant genetics ; OD = optical density ; PCR = Polymerase chain reaction ; RT = room temperature ; T4MBP = Top 4 metal binding protein ; EMP = Exponential Megapriming PCR ; GFP = Green fluorescent protein
