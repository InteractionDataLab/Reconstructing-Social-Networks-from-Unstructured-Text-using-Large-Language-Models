What we pretend with this mathematic model is to represent in a numerical way, the expression of the protein in the cell according with the emission that each concentration of GFP provides. We used linear regression in order to get the values of the constants, using statistical methods that also are applied in the equation. The slope is the value given to b and the intersection is the value for a. y=a/(1+be^(-ct)) be^(-ct)=a/y-1 ln(b)-ct=ln(a/y)-ln(1) c=(ln)(b)-ln(a)-ln(y))/t The graph represent an exponential growth of luminescent that at the same time is restricted the own GFP´s spectrum which only gives green color from 395nm to 475nm, also external factors like the medium in which bacteria growth because is limited.
