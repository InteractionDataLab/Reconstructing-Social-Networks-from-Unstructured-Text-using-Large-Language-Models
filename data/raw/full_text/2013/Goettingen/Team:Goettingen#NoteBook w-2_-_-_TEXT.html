For the homology analysis, we used a web server called microbes online. It can be accessed using the following link: http://www.microbesonline.org/ . We found a conserved hypothetical protein with 65.44 % similarity containing 273 amino acids (822 nucleotides) in L. monocytogenes (dacA).
