Awards Below is a list of links to the pages we have made which nominate us for Special Prizes. Applied Design : We have designed a co-culture analysis system which has several, distinct advantages over the traditional methods. Innovation in Measurement : We have created hardware and software that can be used in conjunction to measure the effects of Parts* on co-cultures. Integrated Human Practices : We communicated with the scientific community and incorporated their feedback into our hardware and software. Software : We have created several programs that create and analyse digital holograms in order to count cells. * The measurements our hardware/software can be used to perform are not limited to characterising Parts and cannot characterise all Parts - we have confirmed that we can see changes that affect growth rate, shape, size or, with some improvement to the hardware, motility.
