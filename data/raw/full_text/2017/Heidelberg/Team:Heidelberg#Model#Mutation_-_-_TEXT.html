In the calculation, \(\Phi_{L}\) is prioritised, so set it to zero to use the number of generations for the calculation. Consider \(L_{S}\) as the number of basepairs that is expected to be mutated. If a part of the sequence you are interested in, is highly conserved choose a lower \(L_{S}\).
