Supported Platforms Lachesis is developed with C++ and Qt and can be compiled & run on: Windows Xp/Vista/7 Debian GNU/Linux It should also work many on other platform that support Qt and mysql, but we do not guarantee. MoDeL is written in Perl and should work on any platform that support perl & mysql. Build & Install MoDeL Download & unpack MoDeL and follow the instructions in INSTALL to install MoDeL. You may also need to install mysql in the process. Lachesis NOTE: We are assuming that you have Qt-sdk on your computer. If not, please refer this page and install it first.
