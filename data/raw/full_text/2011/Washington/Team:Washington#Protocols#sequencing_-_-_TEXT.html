JSXX (depending on the number of samples) Go to Genewiz Customer Login Click Place and order --> Create sequencing order Click Same Day --> Online Form --> Pre-Defined --> Enter # of Samples and then Create New Form Enter your DNA Name (make sure it is not redundant and in order of wells you are submitting) DNA TYpe --> "Plasmid" (click the Type button to autofill the rest, same for all others) DNA Length --> "6-10kb" DNA Conc --> "50" (even if this is not true... thats ok) Primer Name --> "VF2_iGEM" or "VR_iGEM" (if you want forward or reverse, respectively) type "Primers are stored at GENEWIZ" in the comment area Click Save and Next Print out receipt (Send to an advisor and we can print it out) and place labeled samples (JS01 ...
