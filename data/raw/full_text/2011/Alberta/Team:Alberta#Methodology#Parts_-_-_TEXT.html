It is also used in other fungus species. TesA' CDS BBa_K612008 The open reading frame for the TesA' gene. This is a thioesterase from E. coli which has been codon optimized for use in N. crassa. This thioesterase preferably cleaves off sixteen carbon fatty acids from the fatty acid synthase complex, producing free fatty acids. Favourite Parts So far only two parts (the 5'FadD region and the TrpC terminator) have been put in IGEM plasmids for judging. The 5' FadD-UTR was made by using the N. crassa genome as a template in a PCR, BsaI cut sites were added to each end using the PCR primers. The TrpC terminator was synthesized with the codon optimized TesA' and was taken out of that part using PCR. The parts plasmids were checked by using NotI and BsaI digestions and running the DNA on a gel.
