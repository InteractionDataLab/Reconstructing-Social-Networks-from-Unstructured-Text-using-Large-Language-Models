    Every Team has a default template provided. Use it like this: 

    {{<YOUR TEAM NAME>}}
    . You can find and edit it by using this URL: 
    https://2016.igem.org/Template:<YOUR TEAM NAME>

    You can also create new Templates by creating new wiki sites under the Template directory. Example: 

    https://2016.igem.org/Template:LMU-TUM_Munich/Template1

    Although using templates under the Template: Namespace is fine for all practical purposes, it's probably safer to move all your production templates under your own team's namespace, since any logged in user could edit your templates otherwise.

    Images

    To embed an image without the grey padded area around it, write the following code, replacing the text inside <> with actual values. See also the MediaWiki Help pages concerning the topic (Images). [[Image:<FILENAME.EXT>|link=<(optional) HYPERLINK>]|<(optional) WIDTH px>]]

    Columns

    To divide the page into columns of different width, use a
    with the corresponding class (see below). The system is based on Twitter Bootstrap, see Bootstrap Documentation.

    The system uses a grid of 12 columns, so to create a column of 50% width, use the following code: 

    <div class="col-sm-6"><HIER KOMMT DER INHALT REIN></div>

    To create two columns with 3/4 and 1/4 width, respectively, use: 

    <div class="col-sm-9"><INHALT SPALTE 3/4></div><div class="col-sm-3"><INHALT SPALTE 1/4></div>

    All columns are responsive and thus take up more space on smaller screens. To learn how to adjust this behaviour, see the Bootstrap Documentation.
