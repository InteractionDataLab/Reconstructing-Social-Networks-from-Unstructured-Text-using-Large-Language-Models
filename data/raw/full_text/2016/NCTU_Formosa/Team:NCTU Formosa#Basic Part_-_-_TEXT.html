So, we then start to create BioBricks of these three toxins which we have obtained the nucleotide sequences. We provide the four basic parts with the single genes of Hv1a, Sf1a, OAIP, and snowdrop lectin. BBa_K1974001 Hv1a PANTIDE that contains the gene coding for Hv1a. BBa_K1974002 Sf1a PANTIDE that contains the gene coding for Sf1a. BBa_K1974003 OAIP PANTIDE that contains the gene coding for OAIP. BBa_K1974020 Snowdrop Lectin PANTIDE that contains the gene coding for snowdrop lectin. BBa_K1974030 GS linker PANTIDE that contains the gene coding for GS linker.
