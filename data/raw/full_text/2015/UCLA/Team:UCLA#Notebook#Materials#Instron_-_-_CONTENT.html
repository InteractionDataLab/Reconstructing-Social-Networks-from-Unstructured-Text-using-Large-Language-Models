    • Instructions for using the Wu Lab Instron

    starting the machine

    1. Sign into the user sheet (Excel document on the computer)
    2. Make sure that load cell is attached
    3. Turn on the machine
    4. plug in the power cord
    5. the switch doesn't work
    6. wait until "2" appears, then turn on the BlueHill software
    7. you must wait until everything is powered on before adding any attachments!
    8. If the power isn't on, you may overload and destroy the load cell

    Controls and Settings

    • Reset GL: set a new "zero" for strain
    • Return: return to the preset zero
  • Jog: course adjustment
  • red button: emergency shutoff
  • trip screw: shuts off the machine if the load cell touches the screw
  • located on the vertical rail

    • Control mode 1: extension
