    • Software : « Linker Generator 1.0 »

    We have developed a computer tool called « Linker Generator 1.0 ». We are pleased to present it to you. This software allows you to determine the linkers for fusion proteins. Its interface is shown below.

    To use it, you just have to follow the instructions in three steps :

    • 1) install pyton (https://www.python.org/downloads/)
    • 2) download the script (https://github.com/gwils28/LinkerG)
    • 3) run it
