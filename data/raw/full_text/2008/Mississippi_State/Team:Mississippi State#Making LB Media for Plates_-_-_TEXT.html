For every 100ml LB to be made, get 2.5g LB powder. !!!Important!!! Only do this step if making media for plates. For every 100ml LB to be made, get 1.5g Agar. Combine LB powder and 100ml water in 250ml autoclaved flask. !!!Important!!! Only do this step if making media for plates. Add Agar. Cover with foil and autoclave tape. Swirl until dissolved. Autoclave: Fill water in autoclave to top of from lip. Close, set time to 20-25min. It will usually be 40-50min before you can open it. Make sure exhaust is set to Slow(Liquids). After 40-50min., put on heat gloves and slowly open machine. Watch out for steam!!! Remove flask, place on metallic surface until it cools. If making LB Broth, you are done at this point. Close bottle. Don't allow Media to cool too much, when warm to touch, pour into plates. Get 1 plate for every 20-25ml Media. Pour media into each plate until surface is covered. If media remains, distribute evenly among other plates. Allow plates to cool, then flip upside-down. Wait overnight.
