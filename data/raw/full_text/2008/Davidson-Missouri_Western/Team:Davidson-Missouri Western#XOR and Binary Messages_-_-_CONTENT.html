XOR logic

We used XOR logic to build our cryptographic hash function withE. coli. XOR, or eXclusive OR, is a logic gate that produces a value of true if and only if exactly one of its inputs is true.

The truth table for p XOR q (also written as p + q, p ⊕ q, or p ≠ q) is as follows:

Exclusive Disjunction
p q p ⊕ q
T T F
T F T
F T T
F F F
