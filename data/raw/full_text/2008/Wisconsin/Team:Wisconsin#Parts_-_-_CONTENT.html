Biobricks

BBa_K130000 This biobrick contains the srlD gene which is inside the pSB1AK3 biobrick vector. It also encodes for amp and kan resistance. BBa_K130001 This biobrick contains the lipD gene inside the pSB1AK3 biobrick vector as well as kan and amp resistance. BBa_K130002 This biobrick contains the torA export sequence as well as kan and amp resistance. BBa_K130003 This biobrick contains the ycbK export sequence as well as kan and amp resistance. BBa_K130004 This biobrick contains the ycdO export sequence as well as kan and amp resistance. BBa_K130005 This biobrick contains the ycdB export sequence as well as kan and amp resistance.
