Total RNA will be isolated using the E.Z.N.A.™ Bacterial RNA Isolation Kit from Omega bio-tek; cDNA will be synthesized using the qScript™ Flex cDNA Kit from Quanta Biosciences; and we are using the ABI ViiA7 platform along with the KAPA SYBR(R) FAST qPCR Kit from KAPA Biosystems.
